Чтобы запустить сервер:
1.В папке Release запустить файл WPFServer.exe.
2.После запуска следует нажать кнопку "Запустить сервер".
3.Сервер стартует на http://localhost:8080

Чтобы запустить клиент:
1.В папке "Client" следует открыть файл Index.html (сначала запускать сервер).

Исходники находятся в папке SignalRClientServer.
